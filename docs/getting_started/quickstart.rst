Quickstart
==========

.. note::

   This quickstart is being written. In the meantime, :doc:`installation`
   covers setup, and the :doc:`/workflows/index` chapters walk through complete
   analyses from loading to fitting.

This page takes you from a fresh install to your first asymmetry plot and fit in
a few minutes. The steps are:

#. install Asymmetry and launch the graphical interface (:doc:`installation`);
#. load an example dataset;
#. group the detectors and calibrate the balance parameter :math:`\alpha`;
#. plot the asymmetry and run a first fit.

Each step links to its full reference page, so you can stop and read more
wherever you need to.
