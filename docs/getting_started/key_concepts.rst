Key concepts
============

.. note::

   This page is being written. For the underlying physics, see
   :doc:`/explanation/musr_primer`; for feature-by-feature detail, see

Asymmetry organises an analysis around a small number of objects. This page
introduces them — runs and datasets, detector groups, the asymmetry, and the
representations that tie an analysis together — so that the rest of the guide
reads clearly whether you are working in the graphical interface or scripting
against the Python API.
